# Student Collection Manipulator 📚

A simple **Python command-line application** that allows users to manage student records using Python collections such as **lists, dictionaries, tuples, and sets**.

This project demonstrates how CRUD operations (Create, Read, Update, Delete) can be implemented using Python data structures.

---

## 🚀 Project Overview

The **Collection Manipulator** is a menu-driven Python program designed to store and manage student information.

Each student record includes:

* Student ID
* Name
* Age
* Grade
* Date of Birth
* Subjects

Users can add, view, update, and delete records directly from the terminal.

---

## ✨ Features

### ➕ Add Student

Add a new student record with:

* ID
* Name
* Age
* Grade
* Date of Birth
* Subjects

### 📋 Show Records

Display all stored student records in a readable format.

### ✏️ Update Record

Update selected student fields:

* Age
* Subjects

### 🗑️ Delete Record

Remove a student record using the **Student ID**.

### 📚 Unique Subjects

Displays all **unique subjects** from every student record.

### 🚪 Exit Program

Exit the program safely.
